# LoanRisk-News-Analytics

Proyek ini berisi analisis risiko pinjaman dan eksperimen model XGBoost.

**Isi utama**
- `app.py`: skrip utama (jalankan dengan `python app.py`).
- `TA14-XGBoost.ipynb`: notebook analisis dan eksperimen XGBoost.
- `Loan_default.csv`: dataset contoh untuk analisis.

**Persyaratan**
1. Buat virtual environment (direkomendasikan):

```bash
python -m venv .venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
```

2. Install dependency:

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

**Menjalankan**
- Untuk menjalankan aplikasi/script utama:

```bash
python app.py
```

- Untuk membuka notebook:

```bash
jupyter notebook TA14-XGBoost.ipynb
```
